# Rayhan Compass

Beautiful desktop GUI for RayhanDB with purple and white theme.

## Setup

```bash
cd rayhan-compass
npm install
npm start
```

## Features

- Purple and white modern UI
- Radix UI components
- Query editor with syntax highlighting
- Table browser
- Real-time connection to RayhanDB server
- Professional design (no emojis)

## Usage

1. Start RayhanDB server first:
```bash
cd ..
.\RayhanDB-server.exe
```

2. Launch Rayhan Compass:
```bash
cd rayhan-compass
npm start
```

3. Connect using:
- Host: localhost
- Port: 7171
- Username: admin
- Password: rayhan123

Enjoy your beautiful RayhanDB management interface!
